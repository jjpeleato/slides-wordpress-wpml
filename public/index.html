WordPres Multilingual (WPML)

https://wpml.org/es

WPML se trata de un plugin premium que permite realizar de manera totalmente profesional la localización y traducción de sitios web. Puede traducir páginas, entradas, custom post type, taxonomías, menús e incluso textos del tema.

La utilización de WPML no requiere de habilidades técnicas o de programación.

Conociendo WPML

WPML está compuesto por un plugin principal y varios módulos o complementos (denominados add-ons). El plugin principal denominado WPML Multilingual CMS es necesario siempre, mientras que puede instalarse cualquier combinación de módulos para dotarlo de mayores funcionalidades.

No es necesario instalarlos todos.

El módulo más importante es WPML String Translations

Conociendo WPML

WPML Multilingual CMS

Plugin principal. Agrega los controles básicos de traducción y permite la traducción de contenidos.

  • URLs (SLUGS)
  • Páginas
  • Entradas
  • Menús
  • Categorías
  • Opciones SEO

WPML Media Traslations

Funcionalidad para trabajar en varios idiomas los archivos multimedia.

  • Título
  • Leyenda
  • Texto alternativo
  • Descripción

WPML String Traslations

Traducción de todos aquellos textos que no son traducibles por el plugin básico.

  • Título del sitio de WordPress
  • Descripción corta del sitio
  • Widgets
  • Textos panel de administración
  • Textos header y footer
  • Textos de plugins y themes

WPML Translation Management

Permite que el administrador de WordPress gestione el trabajo con un equipo de traductores.

  • Crear usuarios traductores
  • Envío de trabajos a traductores
  • Control del progreso de las traducciones

WPML Sticky Links

Integridad de los enlaces internos para no generar enlaces rotos.

WPML CMS Nav

Elementos de navegación que pueden ser utilizados durante la construcción del sitio.

  • Breadcrumbs
  • Barras laterales de navegación
  • Menús desplegables

WPML All import

Tiene como función exportar/importar el contenido.
